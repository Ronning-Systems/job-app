# Job Application - Functional Requirements

## User Authentication
- Users sign up via LinkedIn or Google
- Password reset capability

## Job Description Archival
- Plain text input
- URL pasting support
- Save and retrieve job descriptions

## Job Tracker
- Status field
- Job URL
- Job Description (saved)
- Location (where it's based)
- Remote capability (checkbox/indicator)
- Applied date
- Notes/Comments
- Response received (checkbox/indicator)

## Resume Management
- Create new resumes
- View existing resumes
- Edit resumes
- Upload resume documents

## Output Tuning
- Personal instructions for resume generation
- ATS screening optimization
- Industry panel feedback/tuning

## Job Stages
- Saved
- Applied
- Phone Screen
- Interview
- Executive Call
- Offered
- Rejected
- Withdrawn
- Closed
